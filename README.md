# Boutique Management System

A complete HTML/CSS/JavaScript based boutique management system for managing dresses, orders, cart, and customer interactions.

## Overview

This is a front-end boutique management system that allows boutique owners to manage products, track orders, handle shopping cart, and provide customer information. The system features a responsive design that works on desktop, tablet, and mobile devices.

## Features

- **Home Page** - Welcome section with product gallery and image slider
- **Login System** - User authentication interface
- **Product Management** - Add new products with name, price, quantity, and category
- **Order Management** - Place and track customer orders
- **Shopping Cart** - Add/remove items and checkout functionality
- **About Us** - Boutique information and services
- **Contact Us** - Customer inquiry form
- **Profile Page** - Complete boutique information and navigation
- **Responsive Design** - Works on all devices (mobile, tablet, desktop)
- **Image Slider** - Auto-playing product gallery with play/pause controls

## Tech Stack

- HTML5
- CSS3 (Responsive design with media queries)
- JavaScript (Image slider functionality)
- Google Fonts (Dancing Script)

## How to Run

1. Download all files maintaining the folder structure
2. Open any HTML file in a web browser (Chrome, Firefox, Edge, etc.)
3. Start from `Home.html` or `Profile.html`
